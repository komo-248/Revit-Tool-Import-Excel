Revit Tool — Import Excel

A one-click Revit add-in that parses an Excel lighting schedule and maps it directly to a Revit schedule via a Dynamo script — exposed through a pyRevit toolbar button and a UI popup. Eliminates manual data entry and copy-paste errors for structured table imports.
